CSS BASICS

CSS

Cascading Style Sheet (CSS) unlike HTML is not a programming language, rather it is a styling language that describes the presentation of a HTML document.

Types of CSS

Inline CSS
This is where a style attribute is applied directly on a single HTML element.
Internal/Embedded CSS
A <"style"> tag is added in the <"head"> section of the HTML document.
External CSS
A separate CSS file is created and using <"link"> tag in <"head"> section the CSS document is linked to the HTML document.
